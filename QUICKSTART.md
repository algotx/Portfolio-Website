# 🚀 Quick Start Guide - AlgoTX Portfolio

## Get Started in 3 Steps

### Step 1: View Locally
```bash
# Simply open index.html in your browser
# Double-click the file or right-click > Open with > Browser
```

### Step 2: Customize (Optional)
Edit `index.html` to update:
- Your contact information
- Project descriptions
- Social media links

### Step 3: Deploy
Choose your preferred hosting:

#### Option A: GitHub Pages (Easiest)
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/yourusername/portfolio.git
git push -u origin main
```
Then enable GitHub Pages in repository settings.

#### Option B: Netlify (Drag & Drop)
1. Go to [netlify.com](https://netlify.com)
2. Drag your project folder
3. Done! Your site is live

#### Option C: Vercel
```bash
npm install -g vercel
vercel
```

## 🎨 Customization Tips

### Change Colors
Open `styles.css` and modify the `:root` variables:
```css
:root {
    --primary-color: #6366f1;  /* Change this */
    --secondary-color: #8b5cf6; /* And this */
    --accent-color: #ec4899;    /* And this */
}
```

### Add Your Photo
1. Add your image to `assets/` folder
2. Update the profile section in `index.html`

### Update Projects
Find the projects section in `index.html` and edit:
```html
<div class="project-card">
    <h3>Your Project Name</h3>
    <p>Your project description</p>
</div>
```
