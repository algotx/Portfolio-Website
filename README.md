# AlgoTX Portfolio Website

## Overview
Professional portfolio website for Ukasha Ahmed showcasing skills, projects, and 10 fully functional niche websites.

## Features
- Modern, responsive design with gradient animations
- Comprehensive About Me section with professional details
- Project showcase highlighting full-stack applications
- 10 fully functional niche websites across different industries
- Interactive contact form
- Smooth scrolling navigation
- Mobile-friendly responsive layout

## Technologies Used
- HTML5
- CSS3 (Custom animations, gradients, flexbox, grid)
- JavaScript (ES6+)
- Font Awesome Icons
